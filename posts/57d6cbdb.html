华强方特 实习 第一周
发表于2018-11-17|更新于2022-08-11|华强方特
|字数总计:1.1k|阅读时长:3分钟|阅读量:

前言

  又好长一段时间没有更新博客了。
  不知不觉中，我已经入职华强方特一个星期有多了。
  月薪3000的实习工资算是非常非常不错的待遇了。
  仲予师兄也直言这样的待遇简直让人眼红。
  我最初还不清楚这算是到了什么程度，直到上周周末去参加师兄的部门聚会，我才知道自己的起步着实很高了。
  师兄的特效部门，很多人入行只有1千多的工资，而且这还是正职的工资。
  所以能够进入如此大的公司，我还是深感幸运的。

方特入职情况

  其实我入职的时候还是非常仓促的。
  可能是因为我不是正规渠道进来的，人事部那边的手续还有很多没有处理好。
  我本来6号就正式开始在公司干活的，但是拖到了9号才正式办理了手续（其实8号就可以办理了，但是我回学校去了）
  我以流程TD的身份安排进了技术部。
  赶进来被苏老板部长安排与另一个刚入职的同事一起负责绑定的工作。
  没想到刚入职的同事竟然是今年毕业的动画专业的师兄，真的是让我大为吃惊（暗自幸运，哪里都有人罩着）
  由于苏老板很忙，而且我们的工作主要是辅助设置部门的绑定工作
  因此，我们主要是对设置部部长丸子负责。

  干活其实并没有想象中那么复杂，主要还是围绕绑定和脚本开发上面。
  绑定部门也会提出很逗超出我能力范围的需求，比如说模型自主碰撞之类的。
  但是总体来说周围的人技术并没有我想象中的高(:з」∠)
  技术部门并不是所有人都在干TD的活，包括软件开发、后台开发、前端开发甚至是摄像机影棚搭建的人员。
  因此其实真正做TD开发的人并不多，比我想象中的要少得多。
  另外我实在没想到会Qt designer 已经算是很强的存在了(:з」∠)
  我开发的时候都没有人可以指导我，最后的状态其实还是自己专研，自己解决问题。
  好吧，其实问题不大，反正和之前的状态差不多。

入职感想

  师兄后面又和我说，其实我的技术水平算是很不错了。
  我现在年轻，缺少的是制作经验、流程经验、项目经验。（总结起来就是工作经验）
  不过作为TD，其实制作的经验的要求没有制作人员的要求那么高。
  更多的是给制作人员提供解决方案，我只要清楚制作的原理和代码编写即可。
  当然，制作也能做好，那就更好了。

  其实我能进方特实习着实是幸运了，几乎就等于进腾讯、阿里的程度了。
  上周周末的时候和师兄的部门一起聚餐，我才发现大家入行的时候是如此的困难。
  大家普遍都是从小公司入行，而且起步工资都是很苦的那种，大家都是从五湖四海过来的，来到这么大的公司还是很不错的。
  虽然自己的技术还行，其实提升的空间还很多。
  而且师兄说的工作经验还是非常重要的，我好需要加快学习进度。
  另外我觉得Houdini 的思想，特效还是非常有意思的，有空还是得多加专研。

总结

  最近事情很多，各种忙，教程暂时都搁置了。
  我需要将PyQt的教程补完，后面继续深入Maya Python API
