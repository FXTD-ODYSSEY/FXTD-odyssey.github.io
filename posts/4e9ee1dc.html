第四十一招 挤出和保持面在一起

  挤出命令按住 shift键配合鼠标右键 往下拖动选择挤出

  Maya2016 之后添加了 ctrl+E 快捷键来实现挤出

  如何实现多个面单独自身挤出

  借助浮动面板可以也可以切换

  还可以 ctrl + shift + 右键 可以弹出标记菜单，选择单独挤出

第四十二招 快速选择集

  快速选择集可以将选择记录下来。
  去到 Create > Sets > Quick Select Set 可以创建快速选择集

  弹出了一个命名窗口，添加命名。

  去到 Create > Quick Select Sets > 可以去选择之前保存下来的 快速选择集

第四十三招 创建字体

  去到 Create > Text 可以创建文字

  点击选项框打开设置窗口

  可以先创建出曲线，方便后期控制
  创建出曲线之后可以到 Surfaces > Bevel Plus 去创建实体模型

  这个方法比起字体面板的 Bevel 有更多的控制属性

注： Maya2018 之后文字生成工具升级了，加入了更多有用的新功能。
