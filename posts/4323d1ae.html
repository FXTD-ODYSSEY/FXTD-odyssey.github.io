10月梗概

  一晃眼10月份就结束了，感觉这个月过得五味杂陈。
  10月初是国庆，是个喜庆的日子。
  10月2日从广州回家，休息整顿三天。
  下午帮新屋搬东西，在满是灰尘的房子里劳累，有轻度的过敏反应。
  从那天开始鼻涕直流，不过病情倒是停滞，不温不火。
  那个时候我就继续利用闲暇的时间看 Python For Maya Artist Friendly Programming 的课程
  后面回学校，带了不少零食月饼。
  我还很清楚10月7号的时候和哥哥在食堂吃麻辣烫，那个时候感觉良好。
  接下来的情况就每况愈下了，我记得10月9日或是10日的时候发烧，整个人累得要死。

  后面我在看B站的时候看到逍遥散人推荐 嘣神纪 这个游戏。
  我还很记得那天不舒服，晚上很晚就起来了，但是睡不着，就在玩这个游戏。
  于是我就入坑了，从此每天晚上回到宿舍就是玩这个游戏，充满了期待。
  直到两周之后游戏更新，我的号被官方永久封禁了，从此退坑。
  （为什么被禁了，可能是我玩得太狂热了，用了两周打到排行榜前十，不过都是肝出来的）

  这14天的时间里，也一直在工作室看PyQt相关的课程。
  写了一些总结，下一个部分我总结一下。
  后面学有所成之后就开始写青年之声的爬虫软件，终于又拾回了当时开发SFML的那种狂热感觉。
  满脑子都是这个事情，不断优化代码达到我想要的效果。

  直到最近10月下旬开始基于青年之声爬虫的基础上开发 Pluralsight 下载器。
  开发成功之后打算开发 Lynda 下载器，遇到了不可解决的问题卡到现在放弃了。

  十月下旬还参加了星耀工大的创业之星评选，同时还组织了党支部生活会议，事情真的是多。
  直到最近这两天终于远赴深圳，完成方特实习和租房子的对接手续。
  这两天回家休养前参加了体侧，跑了1000米，累得我半死，并且又恢复了10月初的流鼻涕状态。
  因为身体状态不好，于是我又开始找游戏消遣，最近这两天在家里休养和去深圳都在玩 我的塔防 手游。
  再次陷入沉迷状态（真的是日了狗了）

十月份回顾

花城汇 网页项目

  这个项目真的是日了狗了，本来期待尽快结束的，结果是甲方不慌不忙不去给他们的甲方做验收。
  直到10月下旬的时候验收出问题了，又要我们去修改，真的是…….
  最大的问题是需要修改流花展馆的外部模型，直到目前为止也没有一个好的修改方案（毕竟建模团队罢工了）
  这边技术上要解决的问题不难，我研究了一晚上就OK了。
  一个是花城汇模型文字显示的调整，客户觉得最初的方案看起来太low了（好想爆粗），要求修改好看点，并且希望做成三维的文字。
  Three.js可以制作三维文字，但是默认字体不支持中文，还需要用fontface.js进行字体转换。
  结果我把微软雅黑转成json之后居然20多M，这大小是不能接受的，于是我想尽量用原先的canvas方案解决问题，目前已经有了初步的效果。
  另外要加入的还有 闪烁报警 、 海心沙box模型 、 海心沙接口对接
  这些都是最近很心烦的事情，还好这些问题都不打，最大的问题还是流花展馆的修改意见上。

10月教程详解


  看起来似乎看了很多教程，关键还是得看消化效果吧。
  另外 Python For Maya Artists - Volume 1 和 Beginning Python for Maya with Chris Zurbrigg 都是四倍速甚至手动加速过完的。
  毕竟讲解的内容基本和 Python For Maya Artist Friendly Programming 教程一致。
  同样都涉及到Python的一些基础应用教学，讲解内容大同小异，所以只是大概看一下哪些不同之处而已。
  Beginning Python for Maya with Chris Zurbrigg 这套教程比较特殊，是用 charcoal Editor（Maya的编辑器插件 - 似乎就是教程作者开发的）进行讲解，不过除了这个也没啥不一样的。
  其他有点意思的教程都有写博客进行记录。
  另外还有四套教程没有完全看完，尚未写博客进行记录。
  Maya白话教程是难得的非常棒的中文教程，主要讲解了Maya的C++插件编写，内容太深，看了前面的部分之后打算先放一放。
  PyQt4 Model View Programming Tutorials是师兄推荐的PyQt教程，应该是Youtube上的教程，讲解PyQt的MVC框架，内容也略深，自己打算再深入学习PyQt，巩固好基础先，所以也放一放。
  Python For Maya Artists - Volume 2 这套是在快速看完 volume 1 之后也是快速看了看，内容结合Maya，还是比较好看懂的，只是看到前半部分有点绕，那天晚上也有点浮躁就点到为止了。
  最近主要在看 pyqt5视频教程全集 python GUI开发 教程，从B站上下载下来的，前面讲解也比较浅，主要针对软件开发，所以和CG领域没太大关系，前面部分也是加速看过去的，难度不大。
  10月份看得教程基本上就是这些。
