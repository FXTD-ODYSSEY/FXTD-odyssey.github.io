еүҚиЁҖ

  жңҖиҝ‘иҝҷдәӣеӨ©еҫҲеӨҡдәӢжғ…пјҢжөҒиҠұеұ•йҰҶиҰҒйҷҶз»ӯејҖе§ӢеҠЁе·ҘдәҶгҖӮ
  е·ҘдҪңе®Өжҗ¬еҲ°4жҘјзҡ„е°Ҹең°ж–№еҺ»дәҶпјҢдёәдәҶеҮҶеӨҮвҖқеҗҺдәӢвҖқпјҢиҝҳеҲ—дәҶдёӘжЎҶжһ¶пјҢе’ҢжҜҸдёӘжҲҗе‘ҳз§ҒиҒҠпјҢзңҹзҡ„жҳҜжөӘиҙ№дәҶеӨ§йҮҸж— и°“зҡ„ж—¶й—ҙгҖӮ
  иҝҷе‘Ёдёүиҝҳжңүе…ҡж”ҜйғЁзҡ„е®Әжі•ж“ӮеҸ°иөӣпјҢиҠұдәҶдёҚе°‘ж—¶й—ҙеҺ»еҮҶеӨҮпјҲе…¶е®һжҲ‘еҸӘжҳҜеҪ“еӨ©еӨҚд№ дәҶдёӢпјҢзңҹзҡ„еҫҲж°ҙдәҶпјү
  д»ҠеӨ©жҜ”иөӣзҫӨйҮҢпјҢиҖҒеёҲеңЁзҘқиҙәиҺ·еҫ—еӣҪе®¶еҘ–еӯҰйҮ‘зҡ„еҗҢеӯҰпјҢйӮЈйҮҢйқўжІЎжңүжҸҗеҲ°жҲ‘пјҢиҺ«еҗҚи§үеҫ—жІ®дё§вҖҰвҖҰ.
  жҲ–и®ёеҚ°иҜҒдәҶдёҠж¬ЎеҶҷзҡ„дјҳи¶Ҡж„ҹпјҢеҸҲиў«жү“зўҺдёҖең°дәҶпјҢжүҖд»ҘжңүзӮ№еӨұиҗҪвҖҰ

  жңҖиҝ‘еңЁзңӢзҡ„ж•ҷзЁӢеҲҶеҲ«жңү

  • Pluralsight - Python for Maya Fundamentals
  • Python For Maya - Artist Friendly Programming

  з¬¬дёҖйғЁе·Із»ҸзңӢе®ҢдәҶпјҢз¬¬дәҢйғЁзңӢдәҶеүҚдёӨйғЁеҲҶгҖӮ
  зңҹВ·жІЎжғіеҲ°дёӨдёӘж•ҷзЁӢзҡ„дҪңиҖ…еұ…з„¶жҳҜдёҖдёӘдәәпјҢжӯӨдәәеұ…з„¶иҝӣдәҶpluralsightпјҒпјҒпјҒ
  з¬¬дәҢйғЁж•ҷзЁӢеҫҲжҳҺжҳҫж—¶й—ҙд№…иҝңдёҖзӮ№зҡ„пјҢйӮЈйҮҢжҺЁиҚҗдҪҝз”Ёpythonдё“дёҡIDE pycharmпјҢйҒӯеҲ°жҲ‘дёӘдәәдёҖдёқдёқзҡ„дёҚеұ‘(:Р·гҖҚвҲ )пјҲжүӢеҠЁж»‘зЁҪпјү
  еҗҺжқҘиҪ¬еҺ»зңӢpluralsightзҡ„ж•ҷзЁӢпјҢеұ…з„¶еҸ‘зҺ°иҝҷдёӘеӨ§зҘһд№ҹиҪ¬дәҶVScodeпјҢиҝҷи®©жҲ‘еҫҲе…ҙеҘӢ(гҒҘпҪЎв—•вҖҝвҖҝв—•пҪЎ)гҒҘ
  дёҚиҝҮеҗҺйқўзңӢдәҶеӨ§зҘһж“ҚдҪңзҡ„VScodeж–№жі•пјҢж„ҹи§үдёҚз¬ҰеҗҲжҲ‘зҡ„дёӘдәәжғ…еҶө(в•Ҙв•Ҝ^в•°в•Ҙ)

VScode python ж”ҜжҢҒ

  е…¶е®һиҰҒи®©VScodeж”ҜжҢҒPythonд»Јз ҒжҸҗзӨәдёҚеӨҚжқӮ
  е®үиЈ…pythonзҺҜеўғпјҢз„¶еҗҺе®үиЈ…VScodeжҸҗдҫӣзҡ„PythonжҸ’д»¶

  иҝҷе°ұеҸҜд»ҘиҪ»жқҫеңЁVScodeеҶҷPythonд»Јз ҒдәҶгҖӮ
  дёӢйқўе°ұжҳҜеҲ©з”Ё VScodeзҡ„intellicense е®һзҺ°иҮӘеҠЁиЎҘе…Ёд»Јз ҒеҠҹиғҪгҖӮ

Maya Python intellicense

  pluralsightзҡ„ж•ҷзЁӢеҗҺйқўжҳҜдёӢиҪҪдәҶanacondaпјҢйҖҡиҝҮanacondaжҸҗдҫӣзҡ„python2.7е№іеҸ°жқҘе®һзҺ°зҡ„гҖӮ
  дҪҶжҳҜеҪ“жҲ‘дёӢиҪҪиҝҷдёӘеҢ…д№ӢеҗҺпјҢжҲ‘еҸ‘зҺ°е®ғеұ…з„¶жңү500MпјҢеҪ“жҲ‘е®үиЈ…е®ҢдәҶд№ӢеҗҺпјҢжҲ‘еҸ‘зҺ°е®ғеұ…з„¶еҚ з”Ё4дёӘGзҡ„зЈҒзӣҳз©әй—ҙпјҢеҺҹи°…жҲ‘зҡ„128GSSDпјҢз©әй—ҙдёҚи¶іпјҢеҚёдәҶеҚёдәҶгҖӮ
  дҪҶжҳҜжІЎжңүpythonзҺҜеўғеҰӮдҪ•е®һзҺ°pythonд»Јз Ғзҡ„иҝҗиЎҢе‘ўпјҹ
  з”Ёи„‘еӯҗжғідёҖдёӢMayaжҳҜжҖҺд№Ҳе®һзҺ°pythonд»Јз Ғзҡ„е‘ўпјҹ
  йҖҡиҝҮеӨ–зҪ‘зҡ„дёҖйҖҡжҗңзҙўпјҢжҲ‘еҸ‘зҺ°mayaе…¶е®һеҶ…зҪ®дәҶpython2.7зҺҜеўғпјҢеҸӘйңҖиҰҒе°Ҷе…¶еј•еҜјеҲ°VScodeжқҘжү§иЎҢпјҢе°ұдёҚйңҖиҰҒйўқеӨ–е®үиЈ…еәһеӨ§зҡ„PythonзҺҜеўғгҖӮ
  жӣҙдҪ•еҶөCGз•Ңзҡ„pythonзүҲжң¬дёҖзӣҙжІЎжңүеҚҮзә§пјҢжүҖд»Ҙз”ЁmayaеҶ…зҪ®жҲ–и®ёдјҡжӣҙеҘҪгҖӮ
  mayaзҡ„pythonзҺҜеўғshellеҗҚз§°дёәmayapy.exe

  иҰҒи®©VScodeж”ҜжҢҒMaya зҡ„ python зј–зЁӢйңҖиҰҒдёӢиҪҪзӣёе…ізҡ„ејҖеҸ‘иө„ж–ҷеҢ…гҖӮпјҲе…ідәҺpycharmиҰҒеҰӮдҪ•и®ҫзҪ®еҸҜд»ҘеҸӮз…§иҝҷдёӘж–Үз« пјү

  и§ЈеҺӢдәҶд№ӢеҗҺпјҢеҸҜд»Ҙе°Ҷdevkitж–Үд»¶еӨ№зҡ„еҶ…е®№иҰҶзӣ–еҲ°Mayaе®үиЈ…зӣ®еҪ•зҡ„devkitдёӯ

  еңЁVScodeзҡ„йҰ–йҖүйЎ№зҡ„jsonеҸҜд»Ҙж·»еҠ дёӢйқўзҡ„д»Јз Ғ - иө„ж–ҷеҸӮиҖғ


  иҝҷж ·е°ұеҸҜд»Ҙе®һзҺ°е’Ңж•ҷзЁӢдёҖж ·зҡ„ж•Ҳжһң

VScode еҝ«йҖҹд»Јз ҒжҸҗзӨә

  еҘҪеғҸд»Јз ҒжҸҗзӨәзҡ„й—®йўҳе·Із»Ҹи§ЈеҶідәҶпјҢе…¶е®һдёҚз„¶пјҢеңЁеҪ“еүҚзүҲжң¬зҡ„VScodeдёӯеҠ иҪҪе·Ёж…ўж— жҜ”пјҢиө·з ҒиҰҒзӯү7гҖҒ8з§’д»ҘдёҠпјҢд»Јз ҒжҸҗзӨәжүҚдјҡеҮәжқҘгҖӮ
  дёәдәҶи§ЈеҶіиҝҷдёӘй—®йўҳпјҢжҲ‘зңҹзҡ„жҹҘйҳ…ж— ж•°зҡ„иө„ж–ҷпјҢиёҸйҒҚдәҶgithubзҡ„еҗ„дёӘе…ідәҺ intellisenceе’Ңautocomplet зј“ж…ўзҡ„й—®йўҳ
  дҪҶжҳҜиғҪеӨҹеңЁж №жң¬дёҠи§ЈеҶіжҲ‘еҪ“еүҚй—®йўҳзҡ„еӣһзӯ”пјҢе°‘д№ӢеҸҲе°‘гҖӮ
  жҲ‘з”ҡиҮізңӢеҲ°дёҖдёӘ2016е№ҙзҡ„й—®йўҳпјҢеҲ°дәҶ2018е№ҙжңүдәәеӣһеӨҚеҮәзҺ°й—®йўҳпјҢе№¶иЎЁзӨәдёӨе№ҙйғҪиҝҳжІЎжңүи§ЈеҶіиҖҢд»ӨдәәйқһеёёеӨұжңӣгҖӮ
  иҝҷеҪ“дёӯзЎ®е®һз»•дәҶж— ж•°зҡ„ејҜи·ҜпјҢз”ҡиҮідёҖеәҰи®ӨдёәжҳҜpylintзҡ„жЈҖжҹҘй—®йўҳпјҢиҖҢиҜ•еӣҫи®©Mayapyе®үиЈ…pylintгҖӮ
  дҪҶжҳҜз»ҸиҝҮз ”з©¶пјҢpylintе№¶дёҚжҳҜж №жң¬еҺҹеӣ пјҢжҜ•з«ҹдҪҝз”ЁдёҚеҗҢзүҲжң¬зҡ„pythonзҺҜеўғиҝҳжҳҜдјҡеҮәзҺ°иҝҷдёӘе·Ёж…ўж— жҜ”зҡ„й—®йўҳгҖӮ
  еҗҺжқҘжҲ‘з”ҡиҮіејҖе§Ӣз ”з©¶VScodeзҡ„CPUи°ғеәҰпјҢе№¶дё”еҸ‘зҺ°дәҶдёҖйғЁеҲҶй—®йўҳзҡ„ж №жәҗгҖӮ

  gifе·Із»Ҹз»ҸиҝҮдәҶеҠ йҖҹеӨ„зҗҶпјҢдҪҶжҳҜйҖҡиҝҮдёҠйқўзҡ„gifпјҢзӣёдҝЎд№ҹиғҪж„ҹеҸ—жҲ‘еҶ…еҝғзҡ„ж— еҘҲдёҺеҙ©жәғпјҢеҰӮжһңжҳҜиҝҷз§ҚйҖҹеәҰпјҢйӮЈд»Јз ҒжҸҗзӨәе°ұеҪўеҗҢиҷҡи®ҫгҖӮ
  еҸҰеӨ–pluralsightзҡ„ж•ҷзЁӢд№ҹжңүжҸҗеҲ°Mayaд№ҹжҳҜиҮӘеёҰд»Јз ҒжҸҗзӨәзҡ„пјҢдҪҶжҳҜеҸҜиғҪжҳҜеӣ дёәжҲ‘зҡ„з¬”и®°жң¬жҜ”иҫғжёЈеҗ§пјҢжҲ‘дёҖж—ҰејҖеҗҜжҸҗзӨәпјҢжҸҗзӨәйҖҹеәҰд№ҹе’ҢVScodeдёҚзӣёдёҠдёӢвҖҰвҖҰ
  еӣ жӯӨиҝҷж ·ж— и§Јзҡ„й—®йўҳпјҢжҲ‘и®Өдёәpycharmд№ҹжҳҜж— еҠӣеӣһеӨ©зҡ„пјҢжүҖд»ҘжҲ‘е№¶жІЎжңүе°қиҜ•дёӢиҪҪpycharmи§ЈеҶій—®йўҳгҖӮ

и§ЈеҶіж–№жЎҲ

  еҗҺжқҘжҲ‘зңӢеҲ°VScodeзҡ„python autoCompleteи®ҫзҪ®йҮҢйқўдёҚд»…д»…еҸӘжңүextraPathsеҸҜд»Ҙи®ҫзҪ®пјҢиҝҳжңүpreloadModulesжқҘеҠ еҝ«еҠ иҪҪйҖҹеәҰзҡ„еҠҹиғҪгҖӮ
  еӣ жӯӨжҲ‘д»ҘиҝҷдёӘдёәзӘҒз ҙеҸЈжүҫеҲ°дәҶи§ЈеҶій—®йўҳж ёеҝғзҡ„githubй—®йўҳ
  иҝҷдёӘй—®йўҳд№ҹжҳҜжңҖиҝ‘иҝҷеҮ еӨ©жүҚжңүдәҶж–°зҡ„зӘҒз ҙгҖӮ

  д№ҹе°ұжҳҜиҝҷжҳҜlanguage serverзҡ„й”…пјҢзҰҒз”ЁJediж–№жЎҲпјҢйҮҮз”ЁеҸҰдёҖдёӘиҜӯиЁҖй…ҚзҪ®пјҢе°ұеҸҜд»Ҙи§ЈеҶій—®йўҳгҖӮ
  жҲ‘е°қиҜ•дәҶдёҖдёӢпјҢзҰҒз”ЁдәҶJediд№ӢеҗҺ VScode дјҡиҮӘеҠЁдёӢиҪҪ Microsoft зҡ„дёҖдёӘиҜӯиЁҖзі»з»ҹпјҢе®үиЈ…е®ҢжҜ•д№ӢеҗҺпјҢеҶҚж¬Ўе°қиҜ•гҖӮ
  и°ўеӨ©и°ўең°пјҢиҝҷж¬Ўеҝ«дәҶйқһеёёеӨҡгҖӮ

  д»ҺgifеӣҫеҸҜд»Ҙж„ҹеҸ—еҲ°пјҢиҷҪз„¶иҝҷдёҖж¬Ўд№ҹйңҖиҰҒloadingдёҖдёӢпјҢдҪҶжҳҜйҖҹеәҰзңҹзҡ„жҳҜиҙЁзҡ„йЈһи·ғпјҒпјҒпјҒ~

жҖ»з»“

  жҲ‘зӣ®еүҚзңҹзҡ„жҳҜеҜ№VScodeжғ…жңүзӢ¬й’ҹпјҢдёҚиҝҮдёҚз®ЎжҖҺд№ҲиҜҙпјҢVScodeд№ҹзЎ®е®һжңүе®ғзҡ„йӯ…еҠӣгҖӮ
  еҰӮжӯӨдёҖжқҘпјҢжҲ‘е°ұеҸҜд»ҘйҖҡиҝҮVScodeпјҢж—ўеҸҜд»ҘеҶҷMELгҖҒVEXжңүеҸҜд»Ҙе®ҢжҲҗPythonзҡ„TDе·ҘдҪң(вҲ©_вҲ©)
