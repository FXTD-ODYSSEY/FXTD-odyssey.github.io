欢迎来到我的个人博客

前言

  很高兴可以在这里与你相遇。
  这个博客是从 2018 年中旬开始了我的更新之路,至今已经有 年了
  目前依然怀揣着巨大的热情和努力在自己的这片小天地间记录变化。

  本篇文章将会不定期更新干货，望大家能够受益。

使用指引

  我的博客是基于前端 Hexo 静态博客框架技术搭建的。
  可以在不依赖服务器的情况下实现博客的分类和搜索。

  博客分左边侧边栏和右边的文章罗列。
  左侧已经将一些相关主题的文章系列整理排列在此。

  扩展图标下面有更多选项可以选择。

  1. 点击标签

  2. 点击分类

  3. 点击归档

  4. 点击友情链接

  5. 点击搜索

注：搜索需要下载搜索索引的文件，随着我的博客内容越来越多，这个文件也越来越大，因此需要等待一段时间的加载。

推荐指引 2019-11-12

友情链接

  博客侧边栏上的友情链接我多年遇到问题的时候搜索答案积累下来的链接
  特别推荐其中的一些博客
  因为目前我自己主研 Maya ，所以推荐 Maya 为主，请见谅~

  • 【Maya】 一半君的总结纸

  一半君的总结纸 博客里面也有一堆大神的链接，我自己也没有验证过，也就不重复推荐了。
  从博客里面的文章记录来看 一半君 也是老前辈了。
  另外博客里面翻译的 How-to 系列真的帮助很大，虽然年代久远，但是Maya的架构真的没有太大变化，完全可用。

  • 【Maya】 Autokdesk ADN 官方博客

  Autokdesk ADN 官方博客 这个博客是自动桌官方技术人员的博客。
  可以说里面的内容是最权威最官方。
  另外博客里很多大佬都是中国人（也可能是华人），希望国人的自研架构能早日面世^_^

个人精选

  • PYTHON QT 开发教程

  这个系列教程是我自己花了一个月时间的准备筹划所编写出来的 Qt for Python 教程系列
  从零到进阶的开发路线，当然由于后续时间紧凑，也没有弄得非常复杂，后期结合Maya等DCC软件的案例太少。
  目前计划重置这个系列的教程，后续将推出视频版，预计将在今年内上线。
  作为粉丝的你，记得催更了。

  • HEXO博客搭建历程

  这个系列记录了我博客的各种变迁与升级
  不忘初心，砥砺前行。

  • 观看笔记 - Python For Maya Artist Friendly Programming

  最用心的教程记录，当时初学 Python Qt ， 连 Maya API 用起来也艰难得很。
  这里面记录了自己很多心路历程，推荐给大家。

博客说明

  修改的内容可以参考我 Hexo博客搭建历程系列

  博客目前挂载在三个地方

  • FXTD-odyssey.github.io
  • FXTD-odyssey.coding.me
  • blog.l0v0.com

  这三个地方同步更新，互相备份。
   github.io 是借助 Github Pages 静态页面挂载的免费服务
   coding.me 是借助 Coding Pages 静态页面挂载的免费服务
   l0v0.com 是我的个人域名。

  评论系统是基于 github issue 的 gittalk 实现的
  因此只有在 github.io 的域名下才可以正确评论，其他挂载服务使用 gittalk 会错误跳转到 github.io 的主页上，对此我也没有办法。
