Windows 效率提升指南
发表于 2020-02-05| 更新于 2020-07-17|效率提升指南
字数总计:3.2k|阅读时长: 10 分钟

前言

  目前主力用 windows 平台，因此推荐 windows 平台的软件为主。
  之前找到一篇文章介绍的工具，这个真的非常受用 少数派

  下面推荐的大部分软件都是这篇文章里面介绍到的。
  大家也可以参考一下少数派文章里面的介绍，我这里着重介绍一些我常用的点

  当然这些效率软件有一定的学习成本，属于熟能生巧，工多手熟的范畴，也不一定适合所有人，大家可以按需安装使用。
  这一期文章原本是去年（2019年） 11月份就想开写的，最后还是折腾了3个月，完全将这些软件都用熟悉了之后再写。
  目前提到的这些工具就是我这三个月主推并一直在使用的工具，我会重点分享自己对这些工具使用感悟。

  这些辅助软件有个缺点就是缺少一个管理软件，如果可以将所有的功能都打包到一起也是挺好的。
  安装起来的话就需要大家一个一个去找来下载。

  另外这些辅助插件的确会一定程度的拖慢了电脑的开机速度，而且会导致一大堆后台图标留在电脑下方
  使用之前需要自行斟酌。

  最近在 B站上找到了一个专注于提升电脑效率的网站，里面介绍很多东西也非常受用 B站 网站

腾讯桌面管理


腾讯桌面管理 官网地址

  可能腾讯系的产品都是有原罪的，只要听到是腾讯出品，必定减分(:з」∠)
  但是这次的桌面管理软件属实弄得很好，而且对于全家桶ptsd患者来说还有独立版本安装的福音。
  这么良心的产品，希望大家不要带着偏见去用~ (少数派也有相关的推荐文章)

  使用也非常简单，只需要安装，就会自动拯救你乱糟糟的桌面。
  而且还可以将文件夹目录映射到桌面的格子上，非常方便。

  这个就是经过整理的桌面，配合下面的 listary 启动器，找东西方便太多了。

listary


listary 官网地址

  listary本质上是一个程序启动器，配合腾讯的桌面整理，找东西如有神助。
  当然网上也有 Wox 启动器 配合 history 来实现 listary 的效果，我目前还没有体验过，不过我觉得 listary 一个软件相较之下更加简单。

快速搜索

  listary 使用也非常简单，按两下 Ctrl 键即可唤起输入框。
  在输入框上输入相应的关键字就可以进行文件搜索，会优先搜索快捷方式桌面相关的目录，而且搜索速度非常快。

  配合上面的桌面整理工具，我可以将软件启动的快捷方式放到快速启动的格子里面，然后通过 listary 来快速打开我想要打开的软件。
  

快速执行命令

  Listary 提供了 4 种不同关键字命令，而且完全可以自定义命令，非常高效。

  比如搜索关键字，可以通过 gg 搜索关键字 直接打开网页进行 Google 搜索
  也可以加入一些 shell 命令来调用系统批处理操作。

文件夹下快速索引

  过去在一个庞杂的文件加下，我们只能通过 windows 提供的首字母进行跳转。
  使用 listary 之后就可以直接输入来在当前目录下进行搜索，而且一些意想不到的地方也支持搜索。

收藏夹支持

  在文件浏览器中的空白区域双击或者单机中键可以打开 listary 菜单
  你可以将一些文件路径收藏到 listary 里面，在这里打开。

Ditto


Ditto 官网地址

  其实少数派的文章已经说得很详细了。
  不仅仅支持文字、还支持图片数据记录，内置 搜索 也可以用 listary 来搜索记录

  只是内置的唤起快捷键按起来不太方便，我不太喜欢，改为 Ctrl + Shift + d

  最近发现 win10 也集成了粘贴板历史的功能，可以通过 win+v 来打开。
  可惜还是差了个搜索搜索功能，也无法通过 listary 来搜索。

QtTabbar


QtTabbar 官网地址

少数派文章推荐

  最开始我也是使用 clover 了，后来用勒 QtTabbar 之后，还是跳不过真香定律。
  经过clover还原了浏览器的体验，但是功能上还是差了点。
  最重要的是可以配合下面要介绍的手势切换文件夹标签，之前亲测 clover 无法切换，非常不便。

  详细的使用介绍 少数派的文章介绍已经非常详细。
  另外还有一个很不错的地方就是添加文件预览功能，非常方便。

  基于 IE 浏览器实现的，因此有时候工具不正常的时候可以打开 windows 的 IE 来重置原样。

  win10 启动 QtTabbar 可以去到 查看>选项 的下拉菜单开启。

  另外 QtTabbar 也提供相对简单的批量重命名操作。

常用操作

  下面就说说常用的一些快捷操作，前面打 * 号是我很常用很方便的

常用快捷键
  • * Ctrl + N 新建标签
  • * Ctrl + Shift + C 复制选中的文件路径
  • * Ctrl + Shift + Z 打开上一个关闭的标签
  • Ctrl + Shift + T 新建一个 txt 文档
  • Shift + F2 重命名当前文件夹
对文件夹
  • 按住 Ctrl 双击左键 - 打开一个新标签并显示
  • 按住 Shift 双击左键 - 打开一个新标签不显示
  • 按住 Alt 双击左键 - 打开文件夹属性
对地址栏上的路径
  • 按住 Ctrl 左键 - 打开一个全新的窗口
  • * 按住 Shift 左键 - 打开一个新标签并显示
对文件夹关闭按钮
  • 左键 - 关闭当前标签
  • 按住 Ctrl 左键 - 关闭其他标签除了当前
  • 按住 Shift 左键 - 关闭整个窗口

（我用下面的手势操作关闭窗口比较多）

对标签拖拽
  • * 将文件拖拽到标签上可以移动文件，按住 Ctrl 拖拽可以复制文件
  • * 将标签拖拽出来可以新建一个全新的窗口

WGestures


WGestures 官网地址

  非常棒的免费手势软件，windows平台专用，用上就爱上了。
  安装完成之后，软件会有很详细的使用说明，配合上面的工具使用就非常省心。

  虽然轨迹支持 中键 和 X键，但是因为各种不方便，我就禁用了。
  摩擦边和触发角也不太习惯，而且还很容易误触，因此也及禁用掉了。
  因此日常操作我就是使用这个软件进行右键滑动~

Quicker


Quicker 官网地址

Quicker 少数派文章

  quicker 非常良心，软件有功能属于付费版本，但是价格非常便宜。
  而且提供免费同步服务，简直白送。

常用操作

  • 截图 OCR
  • 杀进程
  • 朗读文本

  因为上面的工具已经提供了大部分日常的应用，因此我使用 quicker 的频率不是很高
  不过实际上， quicker 提供用户自己制作功能，整个生态比起上面的工具都要好得多
  也可以通过 quicker 给软件指定定制化的操作，只是目前我个人用得还比较少。

  由于 Maya 调试脚本经常需要关闭重启，quicker的杀进程就非常非常好用，quicker 可以获取到程序 id 只需要执行 shell 命令杀进程就可以了。
  当然 WGesture 也提供了强制关停程序的操作，只是手势在 Maya 中使用非常不方便， quicker 提供了另一种舒服的方式。

snipaste


snipaste 官网地址

少数派文章

  snipaste 可以说是 windows 平台上最棒的截图工具了
  以前弄截图不是借助 微信 QQ 就是 借用浏览器的功能。
  snipaste 借助 Qt 框架实现全平台

多层截图 - 方便制作说明

  有时候软件的某些操作在不同的窗口上
  很难通过一张截图说明打开步骤
  这个时候可以借助 snipaste 的 F3 贴图功能
  这个功能可以将之前的截图以悬浮图片的形式展示，就可以将多张贴图放到一起，然后再截一张图说明。
  步骤不多的情况下，再也不需要用 word 文档制作多图步骤了。

取颜色

  无论是网页开发还是如今的工具人，都需要获取某些地方的色块
  snipaste 就提供了全局取色的功能。
  F1 键进入截图只有，截图会有提示，点击 c 键可以复制当前坐标下的像素。
  按 Tab 键可以切换不同颜色码。

ScreenToGif


ScreenToGif 官网地址

  ScreenToGif 是个 github 上的开源项目 github网址
  作为工具人，经常需要介绍工具的使用，使用动图就可以简单扼要讲解清楚。
  过去我都是使用 gifcam 来制作动图的，但是录制 Maya 的时候偶尔会绿屏，而且录制时间不能太长，而且录制出来的动图文件还贼大，试过几十M都有。
  自从用上了 ScreenToGif 感觉动图制作再也不是瓶颈。

capslock+


capslock 官网地址

  这是我最初认识到 autoHotKey 脚本编写的工具，后面我独自研究才发现了新大陆，原来 autoHotKey 这么强大。
  而这个工具实现了花式玩坏无用的大写按键。

  只可惜插件提供的很多功能都和上面的插件重复了，因此我日常使用的情况不会很多。
  当然通过大写键将右手边的按键转移到左手是非常棒的思路。

  另外它也提供了 listary 的启动框，还可以借助 IE 引擎，实现快速调用 js。
  比如 语言轰炸机 或者 简单的算术

  输入上面的 JavaScript 代码，然后按 Capslock + Tab 就可以执行了

  可惜使用上存在一些遗憾
  最后我决定自己动手，丰衣足食，于是就在原作者的基础上进行了一些修改 github仓库

keycastow


keycastow 官网地址

  这款软件用来显示键盘的输入指令，对于教学之类的讲解非常有帮助。
  虽然完全基于windows平台，但是特别小巧，使用方便，而且开源，可以直接查看是否含有恶意代码。

  启动之后右下角会显示键盘输入的所有指令，甚至是鼠标指令也可以显示出来。

  只可惜作者没有将源码编译一个发行版本出来，可以自己编译一个安全的版本，或者下载被人做好的 地址
