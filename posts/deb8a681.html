Windows 效率提升指南
Windows 效率提升指南 （二）

  如今回想起来其实还是那几个经典软件保留在我系统的后台中。

Windows 效率软件汇总

  下面介绍的软件基本都在我上面的两篇效率指南里面有所提及。
  只是有一些不常用的软件被我干掉了，并且这期会将我这些软件常用或是黑科技的功能分享给大家。
  下列介绍的软件都是可以免费下载的软件，大部分是开源可信赖的。

listary


  如果我只能在别人的 windows 上装一个软件。
  那么我肯定会首选 listary
  listary 胜在它简单小巧，却十分简单好用。

  listary 是一革命性的 windows 搜索工具，肩负软件启动器功能。

  安装包不需要网上找，可以直接在软件管家里面安装启动。
  目前管家里面的安装包是 listary 5 的，日常使用没有什么太大的问题。
  不过我还是推荐 listary 6，只可惜 listary6 官方停更了。

listary 6 Beta 官方安装包

  这个是官方最后的波纹了，幸好我还留了一个安装包，欢迎大家安装。目前官方已经没有最新版的下载地址了。

  安装完成之后，双击 Ctrl 唤起全局搜索，搜索速度非常快。

  可以用拼音简写定位软件。

  可以通过简写跳转到对应网站的搜索。

  这些搜索功能都是可以自定义配置的。

  当然也可以自定义系统的命令，实现功能代码的调用。

  对默认的快捷键感到不爽也可以进行自定义配置

  在文件浏览器按鼠标中键可以打开菜单，通过这个菜单可以对相应目录进行收藏。

  这样跳转路径的时候非常方便了。

  以上的功能还只是皮毛，下面两个功能我觉得对日常的工作非常方便。

  使用 windows 自带的文件浏览窗口可以，点击两个文件夹窗口，会自动进行跳转。
  按 Ctrl + G 也可以强制跳转到对应文件窗口上。

  Ctrl + Shift + C 可以复制文件的路径
  如此就可以直接在文件窗口粘贴文件的路径进行打开操作了。

  直接在文件窗口敲键盘即可出发搜索输入框。
  可以直接在文件夹内进行文件搜索和定位，在文件很多的情况下非常好用，比起系统自带的搜索快很多。

同类软件对比

  和 listary 类似的软件还有 wox 和 utools
  wox 是 github 上开源的启动器，国内并不太流行，utools 用的人反而更多。
  utools 则是一款基于 electron 的启动器，优势在于支持多平台，但是安装包会比较大。

  utools 我使用过，默认功能只能搜索系统快捷方式里面的软件。
  无法做到 listary 搜索全局文件，需要结合搜索神器 everything 来实现搜索。
  并且也无法实现文件夹内搜索定位文件，以及文件窗口自动跳转这些核心方便的功能。
  虽然 utools 集成了很多使用的工具包，但是这些功能其实都和 quicker 有所重叠。
  所以经过我的再三权衡，我还是把 utools 给卸载了。

quicker

https://getquicker.net/

  quicker 则是另一个牛逼的神器，而且随着它的发展状态，越来越有 all in one 的发展势头。
  quicker 相对 listary 要复杂一点，否则它就是我的第一推荐了，但是对于小白的简单使用也是非常简单的。
  并且 quicker 借助了社区的优势，通过 共享动作的方式 可以实现无数的可能。
  quicker 是付费软件，但是它的免费版就提供了多端同步的功能，不付钱就能满足绝大多数的使用场景。

  quicker 的表面使用其实相对简单，按住鼠标中键可以打开一个盒子，盒子有相应的功能做触发。
  比如上面我就通过盒子里面的杀进程功能将 Maya 快速关闭。

  你可以在盒子的格子上右键创建一些相应的操作。
  比如点击发送快捷键，发送固定的文本，启动相应的软件等等。
  这些只是最基础的用法。
  更牛逼的是，quicker 使用搭积木的方式让你实现编程。
  当我们在上面的选项中选择 新建组合 动作就可以弹出一个积木窗口。
  你可以通过自己的组合结合一些编程的知识，实现更多更复杂的操作。

  如果你是小白，并不想自己造动作的话，也可以去quicker 的动作库，找广大网页分享的动作。

我的常用动作

杀进程

  我自己加了一些修改实现强制关闭。

剪贴板历史

  这个动作帮我替代掉 Ditto 记录粘贴板历史

截图OCR

  用这个识别的文字非常准。

EVER截图

  这个截完图之后可以钉住图片进行一些列操作。

文本/截图翻译

  这个结合浏览器扩展可以 沙拉查词 可以非常方便查阅不同翻译的信息以及语音

  上面这些动作还只是 quicker 的表层的社区共享功能。
  更多的功能可以去到 quicker 的动作库去查找 链接

WGesture & Quicker

https://www.yingdev.com/projects/wgestures

  WGesture 使用过鼠标手势滑动触发相应快捷键的功能。
  这些功能最新的 Quicker 都已经集成了，quicker 还可以实现多端同步，只是 quicker 需要付费才能解锁这个功能。
  如果不想付钱可以用 WGesture

  通过滑动鼠标可以实现各种快捷键触发。
  在浏览网页和文件的时候非常好用。

  不小心划错了，可以多划几下，识别错误就不会触发任何指令。

snipaste

https://zh.snipaste.com/

  我目前最常用的截图软件。
  可以实现图片钉在桌面上，也可以对图片进行二次的修改。

截图标注

  钉住的图片 滚轮 可以调整大小，按住 ctrl 滚轮可以调整透明度。

取色

  snipaste 可以在屏幕上直接取色，按 shift 可以切换不同表示的颜色值。

腾讯桌面管理

https://guanjia.qq.com/product/zmzl/

  就我个人来说，电脑的桌面通常很少会触碰。
  因为我的应用场景无论是工作敲代码还是日常刷视频，都被各种软件挡住桌面。
  如果要从桌面作为入口进行处理，会带来极大的不便。

  所以我对桌面的要求见简洁清爽，将多余的快捷方式等东西可以用容器将他们全部整理起来。
  不至于让桌面非常凌乱，影响我心情。
  结合上面的 listary 搜索，我可以保留快捷方式到桌面方便我快速搜到。
  同时也不至于让我桌面满屏的快捷方式。

  腾讯桌面管理可以很好地解决我的痛点。

Capslock+ 魔改

Capslock+ 个人魔改 github 仓库

  这个操作是当时看网上一篇关于 AHK 解决中文标点输入的文章学到的，于是我将它的功能融合到 魔改的 Capslock+ 工具里面了。
  简单的意思就是将 打字法 的中文标点输入取消，然后通过 AHK 实现 HotString ，两个英文标点自动替换为中文标点。
  我觉得这个操作用起来真的很舒服，敲代码的时候再也不会因为敲错中文标点报错了。

  解析代码进行连发。

  标点符号自动替换
  自定义指令替换

QtTabbar

http://qttabbar.wikidot.com/qttabbar1

  一款比 clover 还要好用的文件多标签资源管理器。

http://qttabbar.wdfiles.com/local--files/documents/install.html

  根据这里的官方文档安装软件

  完成之后可以在文件窗口的查看选项找到下拉菜单，使用 QTTabBar 就可以开启标签页。

  可以对标签进行诸多的操作。

  也可以直接将文件拖拽到另一个标签页下，默认是移动，按住 ctrl 键就会变成复制。

  自带批量重命名工具，重名也能预览命名的效果，还支持正则表达式等复杂的命名匹配。
  虽然网上也有推荐更为复杂的重命名工具，我个人觉得这个工具已经够用了。


  鼠标悬浮视频和图片还能有预览出现。

  文件夹里面的文件也可以点击图标快速查看。

  基本上日常使用知道上述的就够用了，不过这个工具还有扩展的 API 以及详细的文档。
  从中可以了解更多进阶的用法。

自定义配置

http://qttabbar.wdfiles.com/local--files/documents/summary.html#section_what

  它的配置里面有很多功能都是可以自定义的，具体可以翻阅线上文档。

  比如可以对标签页用各种组合的点击来触发不同的效果。
  其中用户命令可以实现绑定自定义的命令触发。

Scripting API

http://qttabbar.wikidot.com/scripting

  让我感到不可思议的灵活的是这个插件居然有自己的 API
  可以用 js 或者 vbs 来调用。
  只是需要注意这里支持的 js 是用 windows cscript 执行的
  cscript 不支持新版的 es6 标准，也不是 node.js 环境，所以写法会比较受限。

f.lux

https://justgetflux.com/

  屏幕颜色调节，护眼最爱。
  只是当我用惯了之后，很多人看我的屏幕都会很不习惯(:з」∠)

TrayS

https://gitee.com/cgbsmy/TrayS

  当初找到这个主要是为了能够给我美化一下 windows 的任务栏
  当时也找到同类的产品 TaskbarX
  这个也不差，而且也是 Github 开源的，只是它的功能相对繁琐，我还是喜欢 TrayS 简约风。
  而且它也不支持 CPU 占用的显示。

  任务栏图标居中显示，并且可以实时显示 CPU 和 内存占用
  特别是我装了联机编译，有时候 CPU 和 内存占用很高就可以很快看到是哪里被占用了。
  其实它还可以显示 CPU 温度之类的功能，只是我觉得用不上就没有开启。

7+ Taskbar tweaker

https://rammichael.com/7-taskbar-tweaker

  这个工具可以和上面的 TrayS 相辅相成。
  这个软件可以对你的任务栏进一步进行定制化

  安装完成之后的配置里面自定义任务栏图标的不同点击效果。

  在任务栏空白区域双击可以让任务栏自动隐藏，我再双击又可以让任务栏锁定显示。
  有时候远程连接到别人的电脑，我侧边显示的任务栏会有所遮挡，这个时候快速隐藏就很方便。

  空白区域按中键可以设置为任务栏监控插件，可以快速查看当前任务栏上打开的软件。

  还有一些额外的小功能，比如任务栏滚动自动切换软件，只是这些功能我平时用得不多。

ImageGlass

https://imageglass.org/
https://github.com/d2phap/ImageGlass

  windows 自带的图片浏览器非常的一般，很多图片格式都不支持预览。
  而且启动速度也是慢地离谱。

  后来我搜了比较好用的图片预览软件，市面上其实有很多。
  只是有不少有大有臃肿，集合了我很多不需要的功能，个人还是比较喜欢系统原生的简单界面。
  于是综合下来就是 imageglass 最符合我的需求，而且开源目前依然不断在更新维护。

  界面相当简洁，还支持主题切换 https://imageglass.org/themes
  缺点是不支持跨平台。

keepass

https://keepass.info/

  keepass 是一款开源密码管理工具。

  它可以自动帮你生成复杂的密码，极大降低盗号的风险。

  我以前使用密码都是用固定的 3 - 4 个的密码组合。
  而且密码安全性相对简单，估计通用的密码本就有这类似的组合。
  如果黑客真想搞我，那可以尝试用通用的密码本暴力破解了其中一个密码。
  由于我各个账号的密码都相差不大，极其有可能会全部遭殃。

我被裸聊勒索后，看我如何揪出背后的团伙]
可以看看这个大佬是怎么用技术和经验去人肉黑产信息的。

  我自己目前还没有经历过被盗号的情况，不过也曾经被钓鱼网站欺骗过。
  还好发现及时，立刻更改了密码。

  所以经过我的深思熟虑，我决定利用 keepass 这个免费软件来管理我的密码，毕竟数据抓在自己手里是最安全的。

  keepass 的数据库可以通过坚果云的免费 webdav 进行同步。
  手机端可以使用 keepass2android 进行密码同步。

  可能你会觉得密码数据库放到坚果云会不会不安全。
  keepass 数据库也是带安全验证，甚至可以用文件作为密码凭据，而这个凭据可以是任意文件，安全性极高。

bandzip

https://www.bandisoft.com/bandizip/

  bandzip 国内开发的良心软件。
  免费版本不带广告，软件内可以直接预览图片，比起吃相难看的 winrar 要好太多了。

  解压文件的右键菜单可以看到文件预览的信息。

carnac

http://code52.org/carnac/

  屏幕实时显示按键操作，使用简单方便。
  方便用来做操作演示。
