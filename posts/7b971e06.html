Edge 浏览器迁移
发表于2021-03-22|更新于2022-12-14|效率提升指南
|字数总计:1.2k|阅读时长:4分钟|阅读量:

前言

  以前写了一篇关于 Chrome 浏览器推荐的文章 链接
  那个时候我从 360浏览器 迁移到了 QQ浏览器。
  迁移最痛苦的是要重新习惯的收藏夹的分布，不过两个国产浏览器的体验其实大差不差。
  QQ浏览器如果登陆了还能同步安装了的 浏览器插件，这个比 360浏览器还要好点。
  当时我已经听说很多网站都推荐用 微软新推出的 Edge 浏览器。
  我以为就是默认的 win10 Edge 浏览器，最近尝试了才知道原来要另外下载安装包进行安装的。

  最近在研究一些更好的笔记软件 比如 notion 、 wolai 等新式 all in one 软件。
  然后有被 QQ 浏览器古老的 Chromium 内核给气得火冒三丈，感觉 QQ浏览器自发布以来就没有更新过了(:з」∠)。
  网页的体验会有些许 BUG ，让我很难接受。

  所以我也弃暗投明，转到 Edge 上了。
  但是转过来因为之前养成的很多习惯问题，来到 Edge 上发现都没法配置，让我难受了好一会😢
  经过一番摸索，装了好几个辅助的 Chrome 插件总算是还原了大体差不多的体验。

插件安装

Super Drag for Google Chrome

https://chrome.google.com/webstore/detail/super-drag-for-google-chr/dlknjglebgomjjfaijjnebecgjbfjihk

  这个插件可以实现拖拽链接打开一个新的标签。
  而且还多了拖拽文字进行搜索功能

Open Tabs Next to Current

https://chrome.google.com/webstore/detail/open-tabs-next-to-current/gmpnnmonpnnmnhpdldahlekfofigiffh

  Edge 打开新的链接有时候不是在当前标签的右侧，这让我跳转标签变得很麻烦。
  当时在网上搜了一通都搜不到 Edge 改如何配置这个选项，有找到相关的问题，每个正经的回答。
  这个插件完美解决了我的烦恼。

Close All Tabs

https://chrome.google.com/webstore/detail/close-all-tabs/jcokdfogijmigonkhckmhldgofjmfdak

  QQ浏览器可以 Ctrl+Shift+W 关闭当前所有的标签。
  有时候标签多了可以用这个一键关闭，这个插件的快捷键是 alt+W

划词翻译

https://chrome.google.com/webstore/detail/%E5%88%92%E8%AF%8D%E7%BF%BB%E8%AF%91-%E7%AE%80%E5%8D%95%E5%A5%BD%E7%94%A8%E7%9A%84%E5%88%92%E8%AF%8D%E4%B8%8E%E6%88%AA%E5%9B%BE%E7%BF%BB%E8%AF%91%E6%89%A9%E5%B1%95/ikhdkkncnoglghljlkmcimlnlhkeamad

  以前在 QQ 浏览器直接双击就可以翻译单词，但是没法发声。
  这个插件比之前的还要好用，可以跨多个翻译平台，每个平台的语音都可以直接播放。

Separate WIndow

https://chrome.google.com/webstore/detail/separate-window/cbgkkbaghihhnaeabfcmmglhnfkfnpon

  以前使用国产浏览器最棒的点就是可以将视频弹出到一个小窗口播放。
  虽然有个浏览器插件 Picture in Picture Mode 也可以实现类似的效果。
  这个功能我是用之前推荐过的 油猴脚本 来触发的。
  使用之后比起小窗口的好处是原网页依然可以进行滚动，看 Youtube 的时候就可以一边放视频，一边看底下的评论了。
  但是这个小窗口模式无法用油猴脚本来控制播放器。
  于是有些时候我还是希望能将特定的 DOM 元素提取出来放置到小窗口里面进行显示。
  这个插件就更加灵活，同时也能实现我想要的效果。
  国产浏览器里面只有视频弹出这个窗口，这个插件可以自定义任意元素的弹出，只是视频不会有按钮提示，需要手动点选配置弹出的元素。

浏览器功能扩展

用好这些隐藏「小开关」，让 Chrome 浏览器更好用

  Chromium 内核还有些隐藏的小功能，可以通过上面推荐的方式进行开启。

总结

  通过以上插件的辅助，就可以还原大部分国产浏览器的功能。
  还有些就需要改变自己的使用习惯了，比如双击标签页关闭标签，这个在 Edge 里面使用鼠标中键进行关闭。
  点击浏览器的收藏标签需要用鼠标中键打开新的标签页，直接左键会覆盖掉当前页面。
