安卓手机效率提升指南（二）
发表于2020-02-29|更新于2022-07-24|效率提升指南
|字数总计:1.9k|阅读时长:6分钟|阅读量:

前言

安卓手机效率提升指南（一）
  上一篇文章介绍了 谷歌商店 里面的一些便捷操作应用。
  其实也涵盖了日常最常用的 APP 了，基本上我日常调用 APP 都是通过上文提到的 三大 APP 。
  这篇文章则重点着墨在 酷安 - 搞机圣地里面出没的，名声不错的 APP 。
  其实我并不是资深搞机佬，也是最近兴致起来才开始研究，因此我拙劣的认识大家莫见怪。

  下面提到的 APP 可以在 酷安应用市场 可以找到

酷安应用市场

  • 太极
  • 黑阈
  • 轻启动
  • 一个木涵
  • 一触即发

太极

  只要是搞机的人都知道，root了手机才算踏入 安卓 的搞机世界。
  但是随着各大厂商对于手机的各种保护，root手机的成本越来越高了，轻则无法升级手机系统，重则丢了手机的保修。
  而且 root 手机有风险，不仅仅会丢失数据，操作不当还有可能让手机变砖，而且 root 了之后还会被各大银行检测封锁。
  虽说上述种种困难，搞机达人都有办法可以应付的，但是大部分人就没有那么多时间来折腾，而且还要冒着各种风险，实在是难受。

  而太极框架的出现，实现了不 root 手机也可以使用 xposed 框架的效果，达到了给应用开发增强性功能的效果。
  几乎不会被厂商检测也没有 root 手机的各种风险，实在是我等懒人的福音！~

↓↓↓注意↓↓↓
使用太极需要重装 APP ，需要提前备份 APP 的数据，安装之后 app 无法正常刚更新。
并且重装了 微信 之后，在其他 APP 上用不了微信登录，也无法正常分享到微信 ， 需要将其他APP也用太极安装才可以。
想要杜绝上面的问题需要解锁 BootLoader ，安装 magisk 开启 太极阳 的大门。 (相当于要 root 手机)

安装 APP 流程

  这里以 微信 安装到太极举例。

  在太极里面选择添加应用。

  然后勾选微信，点击创建。

  太极会创建一个新的微信安装包，然后需要卸载现有的微信，安装太极提供的安装包即可。

  这里拿了网易云音乐来举例，微信的情况也是一样的。
  卸载微信之前，可以将手机所有的聊天记录备份到电脑端的微信上。
  后续在新装的微信上恢复聊天记录即可，基本不影响到聊天数据。
  当然如果担心出问题，还可以用一些备份软件备份一下，一加就有官方的 一加搬家 app 可以备份手机的数据。

  做完上面的流程，就可以根据太极提供的 xposed 模块安装使用了。
  我使用太极主要在微信的功能增强上，因此下面介绍到的 xposed 模块都和微信有关。
  每次安装完新的模块启用之后，需要强制停止微信重开才有效果。
  另外这些 xposed 模块必须在太极安装了微信之后再安装才有效，否则需要重装。

微X助手

  微信辅助功能的绝对神器。
  含有非常多微信增强的功能。

  • 消息防撤回
  • 自动抢红包(听说会封号，我就不开启了)
  • 自动电脑端登录
  • 各种批量操作

微信斗图

  长按表情代开按钮可以打开表情搜索面板，输入关键字也可以弹出类似 QQ 的参考表情。

微信网页多页面

  这个功能真的是太棒了，以前看公众号、或者微信打开的网页，有收到别人有消息给我。
  以前不懂只能先回复消息，然后找回刚才的链接，后来是将相关的页面弄成浮窗解决。
  现在开启这个模块就可以解决问题了。

微信学英语

  微信所有的聊天信息自带翻译功能。
  实属学习外语的神器。

  翻译用的源可以自行设置，默认可以白嫖谷歌翻译，但是有时候不稳定。
  也可以申请 腾讯 或者 百度 的 API 进行翻译。

黑阈

  黑阈这款软件可以实现在 不 root 状态下对后台常驻的应用进行管理。
  使用黑阈可以自定定时杀死某些后台运行的程序，减少耗电量。
  缺点就是杀死后台之后就无法收到程序的推送，肯定是各有利弊的。

  使用 黑阈 还是不简单，如果不 root 的话，还需要谷歌的 adb 软件来运行 app 内部的 shell 脚本方可正常使用。 adb下载地址
  可以参考这篇百度经验的操作 链接

轻启动

  很多人推荐如果 root 了手机使用 大圣净化，如果不 root 手机就使用 轻启动。
  大圣净化听闻要付费使用，我没有用过，毕竟手机没有 root 。
  轻启动有付费和免费版本，我觉得免费的已经够用了。

  自动点击 APP 启动时候的广告跳过。
  甚至连 youtube 的视频广告也可以自动点跳过。
  平日里后台运行，几乎没有任何存在感，应用跳过广告的时候则会提醒我，非常好用。

一个木涵

  这是一个功能全面的工具箱，名声不错。
  最近新增了一个打包网页为 APP 的功能，挺不错的。
  只是功能挺杂的，我日常使用比较少。

一触即发

  这是一个超强的自动化软件。
  感觉内核可能参考了 安卓自动化神器 auto.js
  不过里面的链接方式则是类似层级式的，和 automate 的节点式还不一样。
  毕竟是国人开发，而且目前还不是很完善，但是未来绝对可期。
  官方有提供脚本，可以通过屏幕识色点击来自动玩支付宝的拍小鸡游戏。
  不过我也不参与支付宝这些杂七杂八的东西里面(:з」∠)

  一触即发的自动化比起 FV悬浮球 的自动化功能要完善复杂很多。
  而且一触即发支持代码转译。
  但是由于复杂，我反而不太喜欢这种层级式的操作，操作起来很累(:з」∠)
  不过由社区有大佬开发了自动刷脚本，可以顺利让我完成很多任务。

总结

  以上就是我两篇关于安卓超好用APP推荐。
  还有一些工具APP以及自己很常用的 APP 没有提到。
  比如图片扫描使用全能扫描王， OCR图片识别使用 白描。
  主要还是分享自己使用手机的一些心得和推荐。
  另外我的手机虽然是一加的，但是还没有认真搞过机，root 过权限。
  可能等5G成熟换新手机，旧手机转备用的时候再来折腾吧。
